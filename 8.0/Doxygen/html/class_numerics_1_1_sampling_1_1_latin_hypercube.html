Detailed Description

A class to perform Latin hypercube sampling (LHS).

Authors: Haden Smith, USACE Risk Management Center, cole..nosp@m.h.sm.nosp@m.ith@u.nosp@m.sace.nosp@m..army.nosp@m..mil

Description:

Stratified sampling is used to generate uniform random numbers by dividing the interval [0,1) into n bins of equal probability, where n is the total number of samples required.

In each iteration:

  1. A random number is generated to select one of the remaining bins.
  2. A second random number is generated to select a value within the chosen bin.
  3. The selected bin is marked as used and will not be selected in subsequent iterations.

This process is repeated until all n bins have been sampled, ensuring that each bin is selected exactly once.
